## Solution 1: In-place

- $TC:O(n)$
- $SC:O(1)$
- 注意本题给出的S是包含了将空格扩展成%20以后所需的额外空间的（甚至有的case都超过了）。
- 而且length表示的是扩展前的字符串的长度。

```java
class Solution {
    public String replaceSpaces(String S, int length) {
        char[] chs = S.toCharArray();
        int index = chs.length - 1;
        for (int i = length - 1; i >= 0; i--) {
            if (chs[i] == ' ') {
                chs[index--] = '0';
                chs[index--] = '2';
                chs[index--] = '%';
            } else {
                chs[index--] = chs[i];
            }
        }
        // String(char[] chs, int offset, int count)
        return new String(chs, index + 1, chs.length - index - 1);
    }
}
```
