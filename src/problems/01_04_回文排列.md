# 01.04. 回文排列 (2021.06.07)

https://leetcode-cn.com/problems/palindrome-permutation-lcci/

## Solution 1: HashSet

- $TC:O(n)$
- $SC:O(n)$
- 判断一个字符串是否是某个回文串的排列，只需要判断该字符串所包含字符的出现次数是否均为偶数，或者只有一个字符出现了奇数次。
- 即该字符串所包含的出现奇数次的字符最多只能有一个。

```java
class Solution {
    public boolean canPermutePalindrome(String s) {
        Set<Character> set = new HashSet<>();
        for (char c : s.toCharArray()) {
            // 如果c出现偶数次，最终会被从set移除
            // 如果c出现奇数次，最终会留存在set中
            if (!set.add(c)) set.remove(c);
        }
        return set.size() <= 1;
    }
}
```

## Solution 2: HashMap

- $TC:O(n)$
- $SC:O(n)$

```java
class Solution {
    public boolean canPermutePalindrome(String s) {
        Map<Character, Integer> map = new HashMap<>();
        int count = 0;
        for (char c : s.toCharArray()) {
            // 一边计算字符c当前的出现次数，一边统计当前的奇数字符
            map.put(c, map.getOrDefault(c, 0) + 1);
            if (map.get(c) % 2 == 1) count++;
            else count--;
        }
        return count <= 1;
    }
}
```
