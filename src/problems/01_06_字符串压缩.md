# 01.06. 字符串压缩 (2021.06.09)

https://leetcode-cn.com/problems/compress-string-lcci/

## Solution 1: Simulation

- $TC:O(n)$
- $SC:O(1)$
- 从左到右遍历字符串S，遍历的同时统计当前字符及其对应出现的次数。

```java
class Solution {
    public String compressString(String S) {
        if (S == null || S.length() == 0) return S;
        StringBuilder sb = new StringBuilder();
        char ch = S.charAt(0);
        int count = 1;
        for (int i = 1; i < S.length(); i++) {
            if (S.charAt(i) == ch) {
                count++;
            } else {
                sb.append(ch);
                sb.append(count);
                ch = S.charAt(i);
                count = 1;
            }
        }
        sb.append(ch);
        sb.append(count);
        if (sb.length() >= S.length()) return S;
        return sb.toString();
    }
}
```

## Solution 2: Two pointers

- $TC:O(n)$
- $SC:O(1)$
- 双指针定位连续相同字符的区间。

```java
class Solution {
    public String compressString(String S) {
        if (S == null || S.length() == 0) return S;
        StringBuilder sb = new StringBuilder();
        int i = 0, j = 0, n = S.length();
        while (i < n) {
            char ch = S.charAt(i);
            while (j < n && S.charAt(j) == ch) j++;
            sb.append(ch);
            sb.append(j - i);
            i = j;
        }
        return sb.length() >= S.length() ? S : sb.toString();
    }
}
```
