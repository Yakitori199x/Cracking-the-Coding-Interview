# 01.09. 字符串轮转 (2021.06.09)

https://leetcode-cn.com/problems/string-rotation-lcci/

## Solution 1

- $TC:O(n)$
- $SC:O(1)$
- 如果s2能够由s1翻转得到，那么将s1中必定存在某个位置，依照这个位置将s1分成两段后再反过来拼接（左右两段交换位置），就能够得到s2。
- 所以，如果我们将s1和其自身拼接，那么必然能在中间的某部分找到s2。同理，如果s2不能由s1翻转得到，那么就不存在这样一个位置，在s1和其自身拼接后得到的字符串里肯定不存在s2。

```java
class Solution {
    public boolean isFlipedString(String s1, String s2) {
        // 首先检查s1和s2的长度，长度不同必然不符合要求
        if (s1.length() != s2.length()) return false;
        return (s1 + s1).contains(s2);
    }
}
```
