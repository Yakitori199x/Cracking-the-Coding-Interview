# 02.07. 链表相交 (2021.06.10)

https://leetcode-cn.com/problems/intersection-of-two-linked-lists-lcci/

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
```

## Solution 1: 2 pointers

- $TC:O(m+n)$
- $SC:O(1)$
- 先用两个指针p和q分别从headA，headB出发遍历A和B两个链表。
- 当其中某个指针遍历到链表尾部时，将其重置到另一个链表的表头继续遍历，直至两个指针指向相同的结点（也可能都是null，即不相交的情况）。

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p = headA, q = headB;
        while (p != q) {
            if (p == null) p = headB;
            else p = p.next;
            
            if (q == null) q = headA;
            else q = q.next;
        }
        return p;
    }
}
```
