## Solution 1: 2-Queues

- $TC:O(1)$ for all operations

```java
class AnimalShelf {

    private Queue<int[]> cats, dogs;

    public AnimalShelf() {
        this.cats = new LinkedList<>();
        this.dogs = new LinkedList<>();
    }
    
    public void enqueue(int[] animal) {
        if (animal[1] == 0) cats.offer(animal);
        else dogs.offer(animal);
    }
    
    public int[] dequeueAny() {
        if (cats.isEmpty() && dogs.isEmpty()) return new int[]{-1, -1};
        else if (cats.isEmpty() || dogs.isEmpty()) return cats.isEmpty() ? dogs.poll() : cats.poll();

        if (cats.peek()[0] < dogs.peek()[0]) return cats.poll();
        else return dogs.poll();
    }
    
    public int[] dequeueDog() {
        if (dogs.isEmpty()) return new int[]{-1, -1};
        return dogs.poll();
    }
    
    public int[] dequeueCat() {
        if (cats.isEmpty()) return new int[]{-1, -1};
        return cats.poll();
    }
}
```
